---

## Technologies Covered

Frontend fundamentals include HTML5, CSS3, responsive design, modern layouts using Flexbox and Grid, and JavaScript (ES6+). Styling is handled using Tailwind CSS for rapid and consistent UI development.

Frontend development is built with React, Vite, React Router, modern state management solutions, and TypeScript to ensure robust and predictable applications.

Backend development uses Node.js with Express or Fastify, RESTful APIs, authentication and authorization using JWT, and secure environment configuration.

Database management is handled using MongoDB with Mongoose for schema design, validation, and performance optimization.

Developer tooling includes Git, GitHub, ESLint, Prettier, Postman, Docker basics, and CI/CD fundamentals.
